# HT Ads Agency — Logo Files

## Mô tả logo

Logo HT Agency gồm 2 phần:
- **Icon mark:** Chữ H và T kết hợp thành monogram, có cánh/fin swoosh màu cyan (#06B6D4) phóng ra từ phía trên bên phải — biểu trưng cho tốc độ, sự bứt phá, tăng trưởng
- **Wordmark:** Chữ "Agency" in đậm, sans-serif, màu đen (#000000)

## Màu sắc

| Element | Hex | Mô tả |
|---|---|---|
| Icon gradient start | #2563EB | Blue |
| Icon gradient end | #06B6D4 | Cyan |
| Wordmark | #000000 | Black |
| Background | #FFFFFF | White (transparent khi dùng PNG) |

## Các biến thể

| File | Kích thước | Dùng cho |
|---|---|---|
| `logo-horizontal.png` | 2560×1094px | Header website, email signature, banner ngang |
| `logo-square.png` | 2560×2560px | Avatar social media, thumbnail, favicon source |
| `logo-stacked.png` | 2560×2560px | In ấn, backdrop, profile hình vuông |

## Hướng dẫn sử dụng

- **Nền trắng/sáng:** Dùng phiên bản màu đầy đủ
- **Nền tối/gradient:** Dùng phiên bản icon gradient + wordmark trắng (cần tạo thêm)
- **Kích thước tối thiểu:** Icon không nhỏ hơn 32px chiều cao
- **Vùng an toàn (clear space):** Tối thiểu bằng chiều cao chữ "A" xung quanh logo
